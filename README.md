# 🍳 Recipe App (JavaScript)

A simple and interactive Recipe App built with JavaScript that allows users to browse, search, and manage their favorite recipes. This project is perfect for learning modern JavaScript, DOM manipulation, and working with APIs.

---

## 📌 Features

- 🔍 Search for recipes by name or ingredients  
- 📖 View detailed recipe instructions and ingredients  
- ❤️ Save favorite recipes  
- 🧾 Add your own custom recipes  
- 📱 Responsive design for desktop and mobile  

---

## 🛠️ Built With

- JavaScript (ES6+)
- HTML5
- CSS3
- Optional: API integration (e.g., recipe APIs)
